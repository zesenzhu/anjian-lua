## If 语句

If 语句是程序中最基本的条件判断结构，用于根据条件的真假来决定程序的执行流程。

### 基本语法

完整的 If 语句结构如下：

```vbscript
If 条件表达式 Then 
    // 条件为真时执行的代码
ElseIf 条件表达式1 Then
    // 条件表达式1为真时执行的代码
Else 
    // 所有条件都不满足时执行的代码
End If
```

### 语法说明

*   If 语句可以包含任意数量的 `ElseIf` 分支，也可以完全不包含
*   `Else` 分支是可选的，通常用作默认处理情况
*   程序从上到下依次检查每个条件，一旦某个条件成立就执行对应的代码块，然后跳出整个 If 结构
*   如果所有条件都不成立且存在 `Else` 分支，则执行 `Else` 中的代码

### 条件判断规则

在 If 语句中进行条件判断时，有以下重要规则：

#### 1\. 关系运算符 vs 逻辑判断

```q
Dim val = true
If val Then 
    TracePrint val
End If

// 上面的代码等同于：
If val = true Then 
    TracePrint val
End If
```

当没有使用关系运算符（如 `=`、`>`、`<` 等）时，系统会自动进行逻辑真假判断。

#### 2\. 真值判断标准

在逻辑判断中，以下值会被视为 `false`：

*   `null`
*   `false`

除此之外的所有值都会被视为 `true`，包括：

```q
// 数字 0 和空字符串也会被判断为 true
Dim val = 0
If val Then 
    TracePrint val    // 这行代码会执行
End If

val = ""
If val Then 
    TracePrint val    // 这行代码也会执行
End If
```

#### 3\. 表（Table）类型的特殊处理

表类型数据无论是否包含元素，都会被判断为 `true`：

```q
Dim val = {null}     // 包含一个 null 元素的表
If val Then 
    TracePrint 1     // 会执行
End If

val = {1, 2, 3}      // 包含多个元素的表
If val Then 
    TracePrint 1     // 也会执行
End If
```

**注意：** 要判断一个表是否为空，应该使用 `Len` 函数检查表的长度：

```q
Dim myTable = {null}
If Len(myTable) > 0 Then
    TracePrint "表不为空"
Else
    TracePrint "表为空"
End If
```

## Select 语句

Select 语句同样用于条件判断，但它更适合处理同一个变量的多种取值情况。

### 基本语法

```q
Select Case 表达式
    Case 值1
        // 当表达式等于值1时执行
    Case 值2, 值3
        // 当表达式等于值2或值3时执行
    Case Else
        // 当表达式不等于任何指定值时执行
End Select
```

### If 与 Select 的使用场景

*   **If 语句**：适合判断多个不同的条件表达式
*   **Select 语句**：适合对同一个变量的不同取值进行分支处理

### Select 语句示例

```q
Dim num = Int(Rnd() * 10)
Select Case num
    Case 1, 2, 3
        TracePrint "随机值在1-3之间"
    Case 4, 5, 6
        TracePrint "随机值在4-6之间"
    Case 7, 8
        TracePrint "随机值在7-8之间"
    Case Else
        TracePrint "随机值是：" & num
End Select
```

### Select 语句的特点

*   每个 `Case` 可以包含一个或多个值，多个值之间用逗号分隔
*   `Case Else` 是可选的，用于处理所有未明确列出的情况
*   程序会依次检查每个 `Case`，找到匹配的值后执行对应代码并退出 Select 结构

## 总结

*   **If 语句**：功能强大，适合复杂的条件判断逻辑
*   **Select 语句**：语法简洁，适合单一变量的多值判断场景
*   两者都支持多分支结构，可以根据具体需求选择使用
