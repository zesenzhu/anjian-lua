# 函数

> 来源: https://res.anjian.com/syntax/函数.html

---

## 函数概述

**函数（Function）** 是编程中用于封装可重复使用代码块的重要结构。通过函数，可以将一组操作打包起来，并通过传递参数和返回值来实现特定功能，从而提高代码的复用性和可维护性。

### 函数的优势

*   **代码复用**：避免重复编写相同的代码逻辑
*   **模块化**：将复杂问题分解为小的、可管理的部分
*   **可维护性**：修改功能时只需修改函数内部代码
*   **可读性**：通过有意义的函数名提高代码的可读性

### 函数 vs 子程序

与**函数（Function）**相似的结构还有**子程序（Sub）**，主要区别如下：

| 特性 | 函数（Function） | 子程序（Sub） |
| --- | --- | --- |
| 返回值 | 有返回值 | 无返回值 |
| 使用场景 | 需要处理并返回结果 | 仅执行操作，不返回结果 |
| 推荐程度 | 推荐使用 | 函数可完全替代 |

在实际使用中，函数可以完全替代子程序，建议统一使用函数结构。

## 函数定义

### 基本语法

```vbscript
Function 函数名(参数1, 参数2, 参数3, ...)
    // 函数代码
    
    // 赋予函数返回值（可选）
    函数名 = 返回值
End Function
```

### 函数命名规则

*   遵循**标识符**命名规则
*   不得与按键内置的函数命令重复
*   建议使用有意义的动词或动词短语命名

### 函数特性

1.  **参数**：
    *   可以有零个或多个参数
    *   参数名在函数内部作为局部变量使用
    *   在按键中定义的函数，调用时参数数量必须与定义时一致（不支持可选参数）
2.  **返回值**：
    *   可以有零个或一个返回值
    *   没有赋予返回值时默认返回 `null`
    *   通过给函数名赋值来设置返回值
3.  **流程控制**：
    *   可以使用 `Exit Function` 强制结束函数执行

## 函数调用

### 调用方式

在按键中有两种调用函数的方法：

#### 1\. 直接调用（推荐）

```vbscript
函数名(参数1, 参数2, ...)
```

这种方式可以获取函数的返回值，是推荐的调用方式。

#### 2\. Call 语句调用

```vbscript
Call 函数名(参数1, 参数2, ...)
```

这种方式可以调用函数，但**无法获取函数的返回值**，不推荐使用。

### 调用示例

```vbscript
Function Countdown(times)
    For i = times To 1 Step -1
        ShowMessage "倒计时：" & i & "秒"
        Delay 1000
    Next
End Function

// 调用函数
Countdown(10)

// 带返回值的函数
Function RandEx(min, max)
    RandEx = Int((max - min + 1) * Rnd() + min)
End Function

// 调用并使用返回值
For i = 1 To 5
    TracePrint "随机数：" & RandEx(1, 10)
Next
```

## 函数返回值

### 返回值机制

1.  **设置返回值**：在函数内部对函数名赋值
2.  **默认返回值**：未赋值时默认返回 `null`
3.  **赋值不终止**：赋值返回值不会终止函数执行
4.  **强制终止**：使用 `Exit Function` 来终止函数

### 重要注意事项

*   **函数名用途**：在函数内部，函数名只能用来赋予返回值，不能作为普通变量使用
*   **返回值类型**：可以返回任何数据类型（数字、字符串、数组、对象等）

### 返回值示例

```vbscript
Function FindItem(key)
    Dim t = {"洛阳", "长安", "开封", "咸阳"}
    For i = 1 To Len(t)
        If key = t[i] Then 
            FindItem = i        // 设置返回值
            Exit Function       // 立即退出函数
        End If
    Next
    FindItem = 0               // 未找到时返回0
End Function

// 使用返回值进行判断
If FindItem("开封") > 0 Then 
    TracePrint "找到了"
Else 
    TracePrint "没找到"
End If
```

## 实用函数示例

### 1\. 数学计算函数

```vbscript
// 计算两个数的最大值
Function Max(a, b)
    If a > b Then
        Max = a
    Else
        Max = b
    End If
End Function

// 计算阶乘
Function Factorial(n)
    If n <= 1 Then
        Factorial = 1
    Else
        Factorial = n * Factorial(n - 1)
    End If
End Function

TracePrint "最大值：" & Max(15, 20)
TracePrint "5的阶乘：" & Factorial(5)
```

### 2\. 字符串处理函数

```vbscript
// 检查字符串是否为空
Function IsEmpty(str)
    If str = "" Or str = null Then
        IsEmpty = True
    Else
        IsEmpty = False
    End If
End Function

// 字符串反转
Function ReverseString(str)
    Dim result = ""
    For i = Len(str) To 1 Step -1
        result = result & Mid(str, i, 1)
    Next
    ReverseString = result
End Function

TracePrint "是否为空：", IsEmpty("测试")
TracePrint "反转字符串：", ReverseString("Hello")
```

### 3\. 数组操作函数

```vbscript
// 数组求和
Function ArraySum(arr)
    Dim sum = 0
    For i = 1 To Len(arr)
        sum = sum + arr[i]
    Next
    ArraySum = sum
End Function

// 查找数组中的最大值
Function ArrayMax(arr)
    If Len(arr) = 0 Then
        ArrayMax = null
        Exit Function
    End If
    
    Dim maxValue = arr[1]
    For i = 2 To Len(arr)
        If arr[i] > maxValue Then
            maxValue = arr[i]
        End If
    Next
    ArrayMax = maxValue
End Function

Dim numbers = {1, 5, 3, 9, 2}
TracePrint "数组和：" & ArraySum(numbers)
TracePrint "最大值：" & ArrayMax(numbers)
```

## 函数最佳实践

### 1\. 单一职责原则

```vbscript
// 好的做法：函数功能单一明确
Function CalculateArea(radius)
    CalculateArea = 3.14159 * radius * radius
End Function

// 不好的做法：函数功能过于复杂
Function ProcessData(data)
    // 验证数据
    // 处理数据
    // 保存数据
    // 发送通知
    // 功能过于复杂
End Function
```

### 2\. 参数验证

```vbscript
Function SafeDivide(a, b)
    // 参数验证
    If b = 0 Then
        TracePrint "错误：除数不能为零"
        SafeDivide = null
        Exit Function
    End If
    
    SafeDivide = a / b
End Function
```

### 3\. 错误处理

```vbscript
Function ReadFileContent(filePath)
    // 错误处理机制
    If filePath = "" Then
        TracePrint "错误：文件路径不能为空"
        ReadFileContent = null
        Exit Function
    End If
    
    // 实际的文件读取逻辑
    // ...
    
    ReadFileContent = "文件内容"
End Function
```

### 4\. 函数文档

```vbscript
// 函数说明：计算两点之间的距离
// 参数：x1, y1 - 第一个点的坐标
//       x2, y2 - 第二个点的坐标
// 返回值：两点之间的距离
Function CalculateDistance(x1, y1, x2, y2)
    Dim dx = x2 - x1
    Dim dy = y2 - y1
    CalculateDistance = Sqr(dx * dx + dy * dy)
End Function
```

## 总结

函数是编程中的重要工具，合理使用函数可以显著提高代码质量。关键要点包括：

1.  **明确目的**：每个函数应该有明确的单一功能
2.  **合理命名**：使用有意义的函数名
3.  **参数验证**：对输入参数进行必要的验证
4.  **错误处理**：考虑异常情况的处理
5.  **返回值**：明确函数的返回值类型和含义
6.  **文档注释**：为复杂函数添加说明文档

通过遵循这些原则，可以编写出高质量、可维护的函数代码。
