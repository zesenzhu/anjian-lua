## 数据类型详解

在按键脚本中，数据类型决定了变量可以存储的数据种类和操作方式。了解各种数据类型的特点和用法是编写高质量代码的基础。

### 空值（Null）

**空值**在按键中用`null`表示，它是一个特殊的数据类型，用来表示**已经定义但未进行赋值**的变量状态。

```vbscript
Dim var
TracePrint var  ' 输出：null
```

#### 空值检测

建议使用`IsNull`命令来判断变量是否为空值：

```vbscript
Dim testVar
If IsNull(testVar) Then 
    TracePrint "变量为空值"
Else 
    TracePrint "变量已赋值"
End If
```

* * *

### 布尔值（Boolean）

**布尔类型**数据只有两种值：`True`（真）和`False`（假），主要用于逻辑判断和条件控制。

```vbscript
Dim isValid, isComplete
isValid = True
isComplete = False

If isValid And Not isComplete Then 
    TracePrint "数据有效但未完成"
End If
```

#### 类型转换规则

**数值型转布尔型：**

*   数值为`0`时转换为`False`
*   其他任何数值都转换为`True`

**布尔型转数值型：**

*   `True`转换为`-1`
*   `False`转换为`0`

```vbscript
Dim num, bool
num = 0
bool = CBool(num)  ' 结果：False

bool = True
num = CInt(bool)   ' 结果：-1
```

* * *

### 数值型（Number）

按键中的**数值型数据**为32位精度的浮点数，可以表示整数（int）和小数（float）。

```vbscript
Dim intValue, floatValue
intValue = 100
floatValue = 3.14159
```

#### 十六进制表示

可以使用前缀`&H`来直接表示十六进制数据：

```vbscript
Dim hexValue
hexValue = &HFF      ' 等于十进制的255
hexValue = &H10      ' 等于十进制的16
hexValue = &HFFFF    ' 等于十进制的65535
```

### 字符串类型（String）

**字符串**是由一系列字符组成的数据，这些字符可以是字母、数字、符号、空格等，用来表示文本信息。字符串必须用双引号`""`括起来。

```vbscript
Dim name, greeting
name = "我是字符串"
greeting = "Hello, World!"
```

#### 字符串转义

字符串中使用`\`作为转义符，转义符后跟特定字母可以表示特殊含义：

  
| 转义符 | 说明 | 示例 |
| --- | --- | --- |
| `\\` | 表示普通`\`字符 | `"符号：\\r\\n"` |
| `\r` | 代表回车 | `"第一行\r第二行"` |
| `\n` | 代表换行 | `"第一行\n第二行"` |
| `\t` | 代表制表符（横向跳格） | `"姓名\t年龄"` |
| `\"` | 在字符串中代表双引号本身 | `"他说：\"你好\""` |
| `\xxxxxx` | UTF-8十进制编码字符（常用于Emoji） | `"\240\159\154\129"` 表示🚁 |

#### 多行字符串示例

```vbscript
Dim multiLine
multiLine = "我是第一行\r\n我是第二行"
TracePrint multiLine
```

**注意：** 由于按键助手的调试输出框无法直接展示换行内容，需要右键点击输出内容选择复制，然后粘贴到其他文本框中查看换行效果。

#### 字符串中的双引号处理

有两种方法在字符串中包含双引号：

**方法一：使用转义符**

```vbscript
Dim word
word = "我是自带\"双引号\"的字符串"
TracePrint word  ' 输出：我是自带"双引号"的字符串
```

**方法二：使用两个双引号代表一个双引号**

```vbscript
Dim word
word = "我是自带""双引号""的字符串"
TracePrint word  ' 输出：我是自带"双引号"的字符串
```

### 最佳实践

*   **空值检查**：在使用变量前先检查是否为空值，避免运行时错误
*   **类型转换**：在不确定数据类型时，先进行类型检查再转换
*   **字符串处理**：合理使用转义符，注意特殊字符的处理
*   **变量命名**：使用有意义的变量名，体现数据类型特征
