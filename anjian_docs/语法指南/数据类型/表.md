## 表数据类型详解

### 基本概念

在按键脚本中，**表（Table）**是一种复合型数据结构，相当于数据的集合，是脚本开发中非常重要和常用的数据类型。

表可以细分为两种主要类型：

*   **数组（Array）**：有序的元素集合
*   **哈希表（Hash Table）**：键值对的无序集合

### 表类型特点

*   **动态扩展**：表内的数据可以随时添加、修改或删除元素
*   **灵活性好**：同一个表可以既包含数组元素，也包含哈希表键值对
*   **支持嵌套**：表的数据可以任意嵌套数组或哈希表，实现复杂的数据结构
*   **类型多样**：表中的元素可以是任意数据类型（null、数值、布尔、字符串、甚至其他表）

💡 **参考资料**：想要了解更多表的特性概念，可以参考 Lua 的 table 类型。

### 基础语法

表由花括号`{}`包围，元素之间使用逗号`,`分割，如果是哈希表，每个元素则使用 `key:value` 进行组合：

*   **数组形式**：`{值1, 值2, 值3}`
*   **哈希表形式**：`{键1: 值1, 键2: 值2}`

#### 定义空表

⚠️ **重要提醒**：与其他数据类型不同，表必须先定义才能进行后续赋值操作。

```vbscript
Dim t = {null}  ' 定义空表时，必须在花括号内写上null
```

#### 基本使用示例

**数组用法：**

```vbscript
Dim t = {123, 456, 789, "其他数组值"}
TracePrint t[1]  ' 输出：123
TracePrint t[2]  ' 输出：456
```

**哈希表用法：**

```vbscript
Dim t = {"name": "Tom", "age": 18}
TracePrint t["name"]  ' 输出：Tom
TracePrint t["age"]   ' 输出：18
```

### 表的取值操作

使用中括号`[]`进行取值：

*   **数组**：填写索引数字（从1开始）
*   **哈希表**：填写键名（字符串）

#### 综合取值示例

```vbscript
' 表可以任意嵌套数组和哈希键值对
Dim t = {123, 456, {"name": "Tom", "age": 18, "list": {789, "其他数组值"}}}

TracePrint t[1]                ' 输出：123
TracePrint t[3]["name"]        ' 输出：Tom
TracePrint t[3]["list"][1]     ' 输出：789
TracePrint t[3]["list"][2]     ' 输出：其他数组值
```

### 数组类型详解

**数组**是有序的元素集合，具有以下特点：

*   元素按顺序存储，有明确的索引位置
*   **索引从1开始**（注意：不是从0开始）
*   使用`[]`和索引数字来访问元素

#### 创建数组

```vbscript
Dim t = {123, 456, 789, "其他数组值"}
TracePrint t[1]  ' 输出：123
TracePrint t[2]  ' 输出：456
TracePrint t[3]  ' 输出：789
TracePrint t[4]  ' 输出：其他数组值
```

#### 修改数组元素

```vbscript
Dim t = {123, 456, 789, "其他数组值"}
t[2] = "修改后的元素"
TracePrint t[2]  ' 输出：修改后的元素
```

#### 添加数组元素

**方法一：直接指定索引**

```vbscript
Dim t = {123, 456, 789, "其他数组值"}
t[5] = "新增元素"
```

**方法二：动态获取索引（推荐）**

```vbscript
Dim t = {123, 456, 789, "其他数组值"}
t[Len(t) + 1] = "新增元素"  ' 更安全的添加方式
```

#### 遍历数组

```vbscript
Dim t = {123, 456, 789, "其他数组值"}
t[Len(t) + 1] = "新增元素"

For i = 1 To Len(t)
    TracePrint "第" & i & "个元素：" & t[i]
Next
```

### 哈希表类型详解

**哈希表**是由键值对组成的无序集合，具有以下特点：

*   使用哈希算法进行数据存取
*   元素没有固定的顺序
*   通过键名快速访问对应的值

#### 创建哈希表

```vbscript
Dim keyName = "class"
Dim t = {"name": "Tom", "age": 18, keyName: "3年2班"}' 可以使用变量作为键名
```

#### 修改哈希表数据

```vbscript
Dim t = {"name": "Tom", "age": 18}
t["age"] = 20
TracePrint t["name"] & "的年龄是" & t["age"]
```

#### 添加哈希表数据

```vbscript
Dim t = {"name": "Tom", "age": 18}
t["class"] = "3年2班"
TracePrint t["class"]  ' 输出：3年2班
```

#### 遍历哈希表

```vbscript
Dim t = {"name": "Tom", "age": 18}

' 遍历初始数据
For Each key, value In t
    TracePrint key & ": " & value
Next

' 添加新数据后再次遍历
t["class"] = "3年2班"
TracePrint "添加新数据后："
For Each key, value In t
    TracePrint key & ": " & value
Next
```

⚠️ **注意**：哈希表的遍历顺序与代码书写顺序可能不一致，这是由哈希算法的特性决定的。

### 实际应用示例

💡以下示例中为了阅读方便，在书写表结构的时候使用了续行符 `_`

#### 学生信息管理

```vbscript
Dim students = {_
    {_
        "name": "张三",_
        "age": 20,_
        "scores": {85, 92, 78}_
    },_
    {_
        "name": "李四",_
        "age": 19,_
        "scores": {90, 88, 95}_
    }_
}

' 访问第一个学生的第二门成绩
TracePrint students[1]["name"] & "的第二门成绩：" & students[1]["scores"][2]
```

#### 配置信息存储

```vbscript
Dim config = {_
    "database": {_
        "host": "localhost",_
        "port": 3306,_
        "name": "mydb"_
    },_
    "features": {"logging", "caching", "backup"}_
}

TracePrint "数据库地址：" & config["database"]["host"]
TracePrint "第一个功能：" & config["features"][1]
```

### 常用操作技巧

#### 1\. 检查表是否为空

```vbscript
Dim t = {null}
If Len(t) = 0 Then 
    TracePrint "表为空"
End If
```

#### 2\. 复制表结构

```vbscript
Dim originalTable = {"name": "Tom", "age": 18}
Dim newTable = {null}

For Each key, value In originalTable
    newTable[key] = value
Next
```
