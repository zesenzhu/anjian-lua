## 逻辑运算符

**逻辑运算符**是编程中用于条件判断的重要工具，用来判断一个或多个条件是否成立。它只返回两种结果：`True`（真）或`False`（假）。

在执行逻辑运算前，系统会自动将参与运算的值转换为布尔类型，然后进行逻辑判断。

### 运算符类型

 
| 运算符 | 说明 |
| --- | --- |
| `And` | **逻辑与**：当且仅当两个条件都为真时返回真，否则返回假 |
| `Or` | **逻辑或**：当任意一个条件为真时返回真，只有当所有条件都为假时才返回假 |
| `Not` | **逻辑非**：返回条件的相反逻辑值，条件为真则返回假，条件为假则返回真 |

### 应用场景

#### 1\. 逻辑条件判断

逻辑运算符最常用的场景是进行条件判断和逻辑对比：

```vbscript
Dim v1, v2
v1 = True
v2 = False

' And运算：只有当两个条件都为真时才执行
If v1 And v2 Then 
    TracePrint "And结果成立"
Else 
    TracePrint "And结果不成立"
End If

' Or运算：任意一个条件为真时就执行
If v1 Or v2 Then 
    TracePrint "Or结果成立"
Else 
    TracePrint "Or结果不成立"
End If

' Not运算：取反操作
If Not v2 Then 
    TracePrint "Not v2 结果为真"
End If
```

#### 2\. 默认值赋值

`Or`运算符还可以巧妙地用于设置默认值，这在处理可能为空的变量时特别有用：

```vbscript
Dim v1, v2, v3
v1 = Null
v2 = "V2的值"

' 利用Or的短路特性：如果v1为空值，则使用v2的值
v3 = v1 Or v2
TracePrint v3  ' 输出：V2的值
```

这种用法的原理是：当`v1`为空值（Null、False）时，`Or`运算符会继续评估右侧的表达式，并返回`v2`的值。
