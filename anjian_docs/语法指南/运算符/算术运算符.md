## 算术与算符

算术运算符对两个数值进行算术运算。

按键中的算术运算符如下

| 运算符 | 说明 |
| --- | --- |
| `+` | 加 |
| `-` | 减、取负(写在数字前则代表负数) |
| `*` | 乘 |
| `/` | 除 |
| `\` | 整除，只返回结果的整数部分 |
| `^` | 乘方 |
| `mod` | 取余数 |

> 使用运算符进行计算时，如果遇到字符串类型数据，则会尝试自动转换成数值型再进行计算，如果转换失败，则在计算时会报错

```vbscript
Dim n1, n2
n1 = 123
n2 = 456
TracePrint n1 + n2

// 字符串类型进行计算会尝试转成数值型
n1 = "123"
n2 = "456"
TracePrint n1 + n2

// 如果无法转换数值型，则会报错
n1 = "123"
n2 = "asd"
TracePrint n1 + n2
```
