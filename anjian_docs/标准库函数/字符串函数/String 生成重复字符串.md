# String 生成重复字符串

> 来源: https://res.anjian.com/standard/string/String.html

---

### 函数名称

String 生成重复字符串

### 函数功能

获取包含指定数量的重复字符组成的字符串

### 支持环境

     
| 系统及平台 | ide（助手） | 安卓（root） | 安卓（无障碍 ） | iOS新版（引擎版本） | iOS旧版 |
| --- | --- | --- | --- | --- | --- |
| 最低客户端版本 | 1.0.0 | 1.0.0 | 5.0.0 | 2.0.0 | 1.0.0 |
| 最低系统版本 | windows 7 | 4.4 | 7.0 | 13.0 | 8.0 |

### 函数语法

ret = String(num, character)

### 函数参数

   
| 参数名 | 参数类型 | 是否必选 | 参数说明 |
| --- | --- | --- | --- |
| num | int | 是 | 重复字符数量 |
| character | string  
int | 是 | 重复的字符或字符串表达式，多字符则取第一个字符，只支持英数、半角符号，不支持中文字符  
传入整数则自动按照ASCIl编码转为字符后拼接 |

### 函数返回

| 返回名 | 返回类型 | 返回说明 |
| --- | --- | --- |
| ret | string | 包含指定数量的重复字符组成的字符串 |

### 代码范例

```sourceCode lua
TracePrint String(3, "a")
TracePrint String(3, "abc")
TracePrint String(3, 42)
TracePrint String(3, chr(42))
```
