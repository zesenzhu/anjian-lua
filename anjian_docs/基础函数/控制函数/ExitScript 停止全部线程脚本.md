# ExitScript 停止全部线程脚本

> 来源: https://res.anjian.com/basic/control/ExitScript.html

---

### 函数名称

ExitScript 停止全部脚本线程

### 函数功能

无条件强制停止所有脚本线程的执行，包含主线程和子线程，支持在子线程内调用

### 支持环境

     
| 系统及平台 | ide（助手） | 安卓（root） | 安卓（无障碍 ） | iOS新版（引擎版本） | iOS旧版 |
| --- | --- | --- | --- | --- | --- |
| 最低客户端版本 | 1.0.0 | 1.0.0 | 5.0.0 | 2.0.0 | 1.0.0 |
| 最低系统版本 | windows 7 | 4.4 | 7.0 | 13.0 | 8.0 |

### 函数语法

ExitScript

### 函数参数

无

### 函数返回

无

### 代码范例

```sourceCode lua
Thread.Start testFunc
Do
    TracePrint "main"
    Delay 999
Loop

Function testFunc()
    dim ranTime = TickCount()
    Do
        TracePrint "testFunc,ranTime:" & TickCount()
        Delay 999
        //子线程执行3秒后停止全部线程，之后主线程不会继续执行
        if TickCount() - ranTime >= 3000 then ExitScript
    Loop
End Function
```
