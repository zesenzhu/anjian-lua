# ColorToRGB分解颜色值为十进制分量

> 来源: https://res.anjian.com/basic/color/ColorToRGB.html

---

### 函数名称

ColorToRGB分解颜色值为十进制分量

### 函数功能

将bgr颜色值分解为r、g、b三个十进制分量

### 支持环境

     
| 系统及平台 | ide（助手） | 安卓（root） | 安卓（无障碍 ） | iOS新版（引擎版本） | iOS旧版 |
| --- | --- | --- | --- | --- | --- |
| 最低客户端版本 | 1.0.0 | 1.0.0 | 5.0.0 | 2.0.0 | 1.0.0 |
| 最低系统版本 | windows 7 | 4.4 | 7.0 | 13.0 | 8.0 |

### 函数语法

ret = ColorToRGB(colorBGR,red,green,blue)

### 函数参数

   
| 参数名 | 参数类型 | 是否必选 | 参数说明 |
| --- | --- | --- | --- |
| colorBGR | string  
int | 是 | 颜色值，格式：BBGGRR，可以为16进制字符串或十进制整数 |
| red | int | 是 | 存储red分量数据的指针变参 |
| green | int | 是 | 存储green分量数据的指针变参 |
| blue | int | 是 | 存储blue分量数据的指针变参 |

### 函数返回

无，结果存储在指针变参中，变参需先自行定义

### 代码范例

```sourceCode lua
Dim red,green,blue
ColorToRGB("FF00AA",red,green,blue)
TracePrint red,green,blue
TracePrint UCase(Hex(RGB(red,green,blue))) //返回bgr十六进制颜色值
```
